# Descrição do Programa

## Introdução
Este documento fornece uma descrição detalhada do programa "Cabeleleira Leila". O objetivo deste programa é gerenciar as operações de um salão de beleza, incluindo agendamentos, cadastro de clientes e serviços oferecidos.

## Funcionalidades
1. **Cadastro de Clientes**: Permite o registro de novos clientes com informações pessoais e de contato.
2. **Agendamento de Serviços**: Facilita a marcação de horários para diversos serviços oferecidos pelo salão.
3. **Controle de Estoque**: Gerencia o inventário de produtos utilizados no salão.
4. **Relatórios**: Gera relatórios detalhados sobre o desempenho do salão, incluindo número de atendimentos e faturamento.

## Tecnologias Utilizadas
- **Linguagem de Programação**: PHP
- **Banco de Dados**: MySQL
- **Servidor Web**: Apache
- **Frontend**: HTML, CSS, JavaScript

## Requisitos do Sistema
- **Servidor**: XAMPP 
- **Navegador**: Compatível Chrome, Firefox, ou Edge

## Instalação
1. Clone o repositório do programa para o diretório do servidor web.
2. Configure o banco de dados MySQL com as tabelas necessárias.
3. Atualize o arquivo de configuração com as credenciais do banco de dados.
4. Inicie o servidor web e acesse o programa através do navegador.
